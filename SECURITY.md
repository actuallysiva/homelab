This repository documents a security-first self-hosted homelab.
Security is treated as a core design principle.

## Reporting a Vulnerability

If you discover a security issue related to:
- Documentation inaccuracies
- Unsafe configuration guidance
- Potential exposure risks

Please do NOT open a public issue.

Instead:
- Contact the repository owner privately
- Provide a clear description of the issue
- Include steps to reproduce if applicable

Responsible disclosure is appreciated.

## Scope

This repository contains:
- Documentation
- Architecture design
- Operational guidelines

It does NOT contain:
- Secrets
- Credentials
- Private keys
- Deployable production code

## Response

Reported issues will be reviewed and addressed
as soon as reasonably possible.
